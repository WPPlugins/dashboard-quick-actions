Admin dashboard quick action icons and links for WordPress

== Description ==
This plugin will add a widget to your WordPress admin dashboard with selected icons, links, and hover effects. You can add icons and links for quick actions such as adding a new post, page, etc. Or you can add icons and links for external things such as your twitter, facebook, etc. 

= Features =
* Select icons, links, link text.
* 170+ Dashicons included.
* Select icon color, background color, size, and hover effect.
* 25+ css3 hover effects included.
* Select link text color, and size.
* Dashboard widget placement options for priority, and side shown.
* Dashboard widget viewing permission option, all user roles available.

== Installation ==

= Automatic Installation =
1. Login to your Dashboard as admin.
2. Once logged in select Plugins -> Add New.
3. Search for or upload Dashboard Quick Actions and activate.
5. Visit Settings -> Dashboard Quick Actions and adjust the settings to your taste.

= Manual Install =
1. Upload dasboard-quick-actions.zip to the wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Visit Settings -> Dashboard Quick Actions and adjust the settings to your taste.
